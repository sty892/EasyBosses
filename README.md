# EasyBosses

Фреймворк для создания кастомных боссов с использованием моделей Blockbench (GeckoLib) для Minecraft (Paper + Fabric).

## Требования
- **Server:** Paper 1.21.1
- **Client:** Fabric 1.21.1
- **Java:** 17+ (рекомендуется 21)

## Структура проекта
- `/plugin` — Плагин для Paper (серверная логика, хитбоксы, управление боссами).
- `/mod` — Мод для Fabric (клиентский рендеринг моделей, анимации).

## Как установить
1. Скопируйте JAR файл плагина в папку `plugins` вашего сервера Paper.
2. Установите клиентский мод на сторону игрока (требуется Fabric API и GeckoLib).
3. При подключении плагин предложит скачать необходимый ресурс-пак (если включен встроенный сервер ресурсов).

## Как создать босса
Минимальный пример `boss.yml` в папке `bosses/my_boss/`:
```yaml
display_name: "Могучий Босс"
base_health: 200.0
bounding_box:
  width: 1.5
  height: 3.0
hitbox_zones:
  head:
    damage_multiplier: 2.0
  body:
    damage_multiplier: 1.0
```
В той же папке должен находиться файл `.bbmodel`. Плагин автоматически экспортирует из него геометрию, текстуры и анимации.

## Известные ограничения
- Система доставки ресурс-паков (resource pack pipeline) находится в экспериментальном статусе.
- Требуется наличие GeckoLib на клиенте.
